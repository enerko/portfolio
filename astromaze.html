AstroMaze

About

AstroMaze is a 2D maze game I developed as part of a course project in Java during my 3rd year at the University of Toronto. In the game, you play as an astronaut lost while studying black hole physics in a distant solar system, racing against time to escape the maze before your oxygen runs out. The game features three difficulty levels, maze customization, user registration and login, and a leaderboard for each level. We followed Object-Oriented Programming (OOP) principles, using Uncle Bobâ€™s Clean Architecture and adhering to SOLID design guidelines. As the gameplay programmer in a group of six, I was responsible for implementing player mechanics, such as input handling, managing the game loop, and designing the user interface. This experience taught me valuable lessons about coding structures, the importance of OOP, and how design patterns can lead to cleaner, more maintainable code.
