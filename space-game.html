2D Space Game

About

I am currently developing a 2D Space Game as part of my journey to learn game development in C++ using SDL. Following Sanjay Madhav's "Game Programming in C++," I aimed to create a project that would allow me to grasp the fundamentals of building a game without relying on an engine. I initially considered cloning classic retro games like Space Invaders or Asteroids but wanted to add my own twist. Thus, I decided to create a 2D side-scroller where the player's goal is to dodge comets, lasers, and planets while surviving for as long as possible. So far, I've successfully implemented the game loop, established a window and renderer, added a side-scrolling environment, and developed systems for input, bullets, collisions, health, UI, and a pause menu.
