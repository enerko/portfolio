Mars Tower Defense

About

Mars Tower Defense is a grid-based tower defense game where the player must defend the base from waves of robot enemies.

Responsibilities

- Built tools for artists - Grid system - Buying, building, deleting turrets - Enemy management with object pooling - Integrating art, music, and vfx
